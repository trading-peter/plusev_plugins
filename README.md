# Plugins for PlusEV

**For educational purposes only! Use at your own risk!**

## Usage

- In PlusEV go to `Settings` (Avatar menu at the top right).
- In the `User` tab (= user specific settings) scroll to the **Plugins** section.
- Look for the setting titled **Plugin Repositories** and click the plus-icon.
- Enter this URL and click **Add**: `https://github.com/trading-peter/plusev_plugins/raw/refs/heads/main/repo.json`

That's it. You will see any available plugins in the plugin settings of the respective app. For example, in `Settings` -> `Data Sources` -> Button `Plugins`.
